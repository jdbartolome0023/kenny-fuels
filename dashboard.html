Optimisations This Week

Google Ads and account actions taken Jun 19 – Jun 25, 2026.

✓

Google Shopping Readiness Audit Completed

A full audit of the Kenny Fuels Merchant Center feed was completed using live GMC data. Key findings: 1,160 products already approved for Shopping in Ireland, 18 products disapproved (5 missing price, 3 missing image, 1 promotional title, 1 image overlay), and 645 product images below Google's minimum 500x500px requirement. A detailed product-by-product URL list was prepared and sent to Graphedia, with the image fixes grouped into a Google Sheet. A separate set of 9 products (healthcare, tobacco, and kerosene variants) was identified for exclusion via GMC feed rules rather than Graphedia action. The audit report is live at the link below. Target: all issues resolved ahead of the September 1 Shopping campaign launch.

✓

HHO tROAS Assessment: Uncapped Maintained

Home Heating Oil has been running on Maximize Conversions uncapped since May. A five-week ROAS assessment was completed this week: weekly ROAS readings of 12.7x, 17.1x, 29.6x, 12.5x, and 19.3x show strong but volatile performance. The landing page order system went live on June 17, and there are only a limited number of post-change data points so far. Decision: keep Maximize Conversions uncapped. Introducing a tROAS target now could unnecessarily constrain a campaign that is already delivering strong results. Next review: July 21.

✎

Conversion Value Rules Confirmed Active

All conversion value rules were verified as active via the Google Ads API. The rules apply revenue multipliers to conversion signals based on campaign: HHO conversions are valued at €350 (base €90 x 3.89), Brand Search conversions at €220 (base €90 x 2.44), Gas Bottle base value €90 (no multiplier). Solar conversion value is pending the average residential installation cost from Ciaran, which will be used to set an accurate value once received.
